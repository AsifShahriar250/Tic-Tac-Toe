# Tic-Tac-Toe
Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. This project is perfect for beginners to understand DOM manipulation, event handling, and basic game logic.

 Features
. Two-player mode (Player O vs Player X)
. Reset game functionality
. Start a new game anytime
. Winner announcement display
. Clean and responsive UI

 Technologies Used
. HTML5 – Structure of the game
. CSS3 – Styling and layout
. JavaScript  – Game logic and interactivity

 Project Structure:
  tic-tac-toe/
   index.html     -Main HTML file
   style.css      -Styling file
   app.js         -Game logic

 How to Run the Project:
 link: https://tic-tac-toc-asif.netlify.app/


 Game Rules:
  The game is played on a 3×3 grid
  Players take turns:
    Player 1 → O
    Player 2 → X
  The first player to align 3 symbols in a row (horizontal, vertical, or diagonal) wins
  If all boxes are filled and no winner → it's a draw
